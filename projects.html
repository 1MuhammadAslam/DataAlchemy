HR Analytics Dashboard

Employee Attrition Analysis & Prediction

Power BI Python Logistic Regression Data Visualization Excel

Overview

This project involved analyzing employee data to understand attrition patterns and predict future resignations. The goal was to help HR departments identify at-risk employees and understand the key factors contributing to turnover.

Key Features

  • Interactive Power BI dashboard with attrition trends by department, role, and tenure
  • Logistic regression model to predict resignation likelihood
  • Key driver analysis identifying top factors influencing attrition
  • Employee segmentation by risk level
  • Cost analysis of attrition impact

Results

The dashboard helped identify that employees with 3-5 years tenure in certain departments had 40% higher attrition rates. The prediction model achieved 82% accuracy in identifying employees who would leave within 6 months. This enabled proactive retention measures that reduced attrition by 15% in the next quarter.

Airbnb Revenue Prediction

London Listings Revenue Estimation Model

R Python Tableau Regression Analysis Data Cleaning

Overview

This project analyzed London Airbnb listings to build a predictive model for estimating potential revenue. The model helps hosts optimize their pricing and listing features to maximize earnings.

Methodology

  • Collected and cleaned data from Airbnb's London listings
  • Feature engineering to create meaningful predictors
  • Multiple regression approaches tested (linear, ridge, lasso)
  • Geospatial analysis of location impact
  • Interactive Tableau dashboard for exploration

Key Findings

The model revealed that beyond location and price, the number of photos and response rate significantly impacted revenue. Listings that responded to 90%+ of inquiries within an hour earned 22% more on average. The final model explained 76% of revenue variance (R² = 0.76).
