HR Analytics Dashboard

Employee Attrition Analysis & Prediction

Power BI Python Logistic Regression Data Visualization

Overview

Analyzed employee data to understand attrition patterns and predict future resignations. The goal was to help HR departments identify at-risk employees and understand key factors contributing to turnover.

Key Features

  • Interactive Power BI dashboard with attrition trends
  • Logistic regression prediction model
  • Key driver analysis
  • Employee segmentation by risk level

Results

Identified departments with 40% higher attrition rates. Prediction model achieved 82% accuracy, enabling proactive measures that reduced attrition by 15%.

Airbnb Revenue Prediction

London Listings Revenue Estimation Model

R Python Tableau Regression Analysis

Overview

Analyzed London Airbnb listings to build a predictive model for estimating potential revenue. The model helps hosts optimize pricing and listing features to maximize earnings.

Methodology

  • Collected and cleaned Airbnb listing data
  • Feature engineering for predictors
  • Multiple regression approaches tested
  • Interactive Tableau dashboard

Key Findings

Response rate and photo count significantly impacted revenue. Listings with quick responses earned 22% more. Final model explained 76% of revenue variance (R² = 0.76).
