HR Analytics Dashboard

Employee Attrition Analysis & Prediction

Power BI Python Logistic Regression Data Visualization

Overview

Analyzed employee data to understand attrition patterns and predict future resignations. The goal was to help HR departments identify at-risk employees and understand key factors contributing to turnover.

Key Features

  • Interactive Power BI dashboard with attrition trends
  • Logistic regression prediction model with 82% accuracy
  • Key driver analysis identifying top attrition factors
  • Employee segmentation by risk level
  • Automated monthly reporting system

Results

Identified departments with 40% higher attrition rates. The predictive model enabled proactive retention measures that reduced attrition by 15% within 6 months. Dashboard adoption led to 30% faster decision-making in HR department.

Airbnb Revenue Prediction

London Listings Revenue Estimation Model

Python Tableau Regression Analysis Feature Engineering

Overview

Developed a predictive model for London Airbnb listings to estimate potential revenue, helping hosts optimize pricing and listing features to maximize earnings. Analyzed over 50,000 listings across Greater London.

Methodology

  • Collected and cleaned Airbnb listing data using Python (Pandas)
  • Feature engineering including geospatial features and amenity analysis
  • Tested multiple regression approaches (Linear, Ridge, Random Forest)
  • Created interactive Tableau dashboard with revenue heatmaps
  • Developed pricing recommendation engine

Key Findings

  • Response rate and photo count significantly impacted revenue (22% premium for quick responses)
  • Location premium varied up to 300% between neighborhoods
  • Optimal photo count identified as 12-18 photos per listing
  • Final model achieved R² = 0.76 on test data
